# 小技巧

- 单击文件 临时打开，可以被替换
- 双击文件 正常打开
- 添加 debugger 断点
- 注释折叠 使用`//#region`开头，`//#endregion`结束
- 鼠标放在 css 选择器上，会自动计算 权重
- 鼠标放在 关键字 上，会提供 MDN 地址

## 输入

- 输入lorem 生成一段英文
- 输入lorem5 生成5个单词的英文

- div.类名 => 该类名的div
- div#id => 该ID的div

- 汉字转拼音 <https://www.aies.cn/pinyin2.htm>
- 解码工具 <https://www.wetools.com/url-encode>
- 字体定制工具: <https://www.iconfont.cn/webfont>

## React

 组件名 / 回车 => <组件名 />
