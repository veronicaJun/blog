# 重排和重绘

1. 重排reflow: 需要重新计算元素的**位置**及**尺寸**
2. 重绘repaint: 需要重新绘制元素的**样式**
3. 如何优化?
    1. 合并多次对DOM和样式的修改，然后一次处理掉
    2. 脱离文档流
        1. 隐藏元素，应用修改，重新显示
        2. 使用文档片段(document fragment)在当前DOM之外构建一个子树，再把它拷贝回文档。
        3. 将原始元素拷贝到一个脱离文档的节点中，修改节点后，再替换原始的元素。
    3. 减少访问元素属性的次数
    4. 开启 GPU 加速: 使用css3硬件加速
        1. transform
        2. opacity
        3. filters
        4. Will-change
