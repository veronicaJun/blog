# nvm

## 一、介绍

nvm全称 **Node Version Manager** 顾名思义它是用来管理node版本的工具，方便切换不同版本的 Node.js

## 二、使用

nvm 的使用非常的简单，跟 npm 的使用方法类似

## 2.1 下载安装

首先先下载nvm，下载地址<https://github.com/coreybutler/nvm-windows/releases>
选择 **nvm-setup.exe** 下载即可(网络异常的小朋友可以在资料文件夹中获取)
